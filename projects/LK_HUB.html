I designed and developed the full LK-HUB platform from scratch for Liberating Knowledge (liberating-knowledge.com) — a media literacy organization focused on accelerating capacity building across MENA. The platform handles multi-division content management, news publishing, project showcasing, and an admin console, all built with a modern Next.js stack deployed on Hostinger.

About Liberating Knowledge

A media literacy and knowledge platform focused on empowering individuals through education and innovative solutions. Structured around 5 divisions — Education, Communication, Industry, Governance, and Policy — all managed through the LK-HUB platform.

Technical Architecture

  • Next.js 16 App Router with SSR
  • React 19 component library
  • MongoDB for dynamic content
  • JWT authentication with bcryptjs
  • Custom Python SFTP deployment script

Key Features Built

  • Multi-division content management
  • Rich-text article/news editor (Tiptap)
  • Admin dashboard with auth middleware
  • Dynamic project & stats display
  • Asset upload API
  • Mobile-responsive design

Tech Stack Details

  • Framework: Next.js 16.1.6 (App Router, standalone)
  • Frontend: React 19 + Framer Motion
  • Styling: Vanilla CSS + styled-jsx
  • Typography: MOKOTO, Avenir Next Arabic, Playfair Display, Inter
  • Database: MongoDB
  • Auth: JWT + bcryptjs
  • Deployment: Hostinger + Phusion Passenger

Design System

Primary colors: burgundy (#3d0000) + gold (#a57c30). Custom CSS variables. Glassmorphism effects. Premium animations via Framer Motion.

Deployment & Infrastructure

Hosted on Hostinger shared hosting via Phusion Passenger. Custom Python SFTP deployment script (deploy_to_server.py) for automated builds and server restarts. Next.js standalone output mode for efficient shared hosting.
