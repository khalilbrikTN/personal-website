Project Overview

Cross-API Bibliographic Data Integration System
Developed an automated system to enhance the AUC Library's catalog by integrating bibliographic metadata from HathiTrust and Open Library. The project aimed to enrich existing library records with standardized MARC XML data, full-text access links, and missing identifiers (OCLC, LCCN, ISBN).

Problem Statement

The AUC Library catalog contained thousands of records with incomplete metadata and missing digital access information.

  • Incomplete Records: Many library records lacked standardized identifiers.
  • Manual Processing: Enriching records manually was time-consuming and error-prone.
  • Digital Access: No systematic way to link physical holdings to digital versions.
  • Solution: Automated API-driven metadata enrichment pipeline.

Technical Implementation

Built a multi-stage data processing pipeline using Python and external APIs.

  • Data Cleaning: Regex-based extraction and normalization of identifiers.
  • API Integration: HathiTrust and Open Library REST APIs for metadata retrieval.
  • Batch Processing: Divided 60,000+ records into manageable sub-batches.
  • MARC XML Extraction: Parsed and stored standardized bibliographic metadata.
  • Data Persistence: Excel/CSV output with comprehensive matching statistics.

Key Results

Successfully processed the entire library catalog and enriched thousands of records.

  • Scale: Processed 60,000+ library records across multiple batches.
  • MARC XML Retrieval: Successfully extracted structured metadata for matching records.
  • Digital Access: Identified full-view and preview links for digital versions.
  • Identifier Enrichment: Cross-filled missing OCLC, LCCN, and ISBN values.
  • Automation: Reduced manual metadata entry workload significantly.

Tools & Technologies

Leveraged modern Python libraries and APIs for bibliographic data processing.

  • Python Libraries: Pandas, Requests, PyMARC, XlsxWriter
  • APIs: HathiTrust Bibliographic API, Open Library Books API
  • Data Processing: Regex, JSON parsing, batch processing
  • Environment: Google Colab for development and execution

Impact & Learnings
